La cartella "taperecorder" contiene tutti i file necessari a far funzionare il 
grammofono con XAMPP (MySQL + Apache).

Installazione:
-------------

> Posizionare la cartella: "taperecorder", all'interno della cartella: "htdocs"
  di XAMPP.
  
> creare un nuovo database tramite phpmyadmin di nome: "phi"

> creare una nuova tabella di nome: "phi_tape" utilizzando il file: "dump.sql"
  allegato.
  
Utilizzo:
-------------

> Avviare il server e il dbms

> Aprire il browser ed usare il link: http://127.0.0.1/taperecorder/taperecorder.php
